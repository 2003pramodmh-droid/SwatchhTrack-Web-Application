# SwatchhTrack-Web-Application
A web-based complaint management system that enables citizens to report sanitation issues and helps authorities track and resolve them efficiently.

SwatchhTrack – Smart Sanitation Complaint Management System

Project Description

SwatchhTrack is a web-based complaint management system that allows citizens to report sanitation issues such as garbage overflow, unclean streets, and waste management problems. The system helps authorities track, assign, and resolve complaints efficiently through a centralized dashboard.

Features

* User Registration & Login
* Raise Complaint with Description & Location
* Upload Images for Proof
* Track Complaint Status
* Admin Dashboard
* Assign Complaints to Staff
* Update & Close Complaints
* Role-Based Access (User/Admin)

Tech Stack

* Frontend: HTML, CSS, JavaScript / Bootstrap
* Backend: Java, Spring Boot
* Database: MySQL
* Build Tool: Maven
* Server: Tomcat

Installation & Setup

1. Clone the repository
git clone https://github.com/your-username/swatchhtrack.git

2. Open in IDE (Eclipse/IntelliJ)

3. Configure MySQL database in `application.properties`

4. Run the project:- mvn spring-boot:run

5. Access at :- http://localhost:8080
